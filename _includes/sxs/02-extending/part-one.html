Part one Customize a card with theme settings

Over on the opposite side of the screen you'll see a USWDS Card component. Let's customize this card a bit for our site. Weâ€™ll start by using theme settings to make simple customizations.

View available theme settings.

Visit the USWDS Card component settings table to see a list of available settings. Settings allow you to apply a global theme to USWDS components and styles without modifying CSS. Today's we're going to give our card a light golden look.

Customize border settings

Let's start but updating the border to remove the border and give the card a rounder border radius.

We want to give our card... TKTKTKTKTKTKT

description _theme/_uswds-theme.scss

@use "uswds-core" with (
  ...

  $theme-card-border-radius: 2,
  $theme-card-border-width: 0
);

Customize type settings

Next lets use the available font and typeset settings to customize how our card information is presented.

description _theme/_uswds-theme.scss

@use "uswds-core" with (
  ...
  $theme-card-border-radius: 2,
  $theme-card-border-width: 0,
  $theme-card-font-family: "serif"
  $theme-card-header-typeset: "sans", "xl", 2.5

);

Save, wait for he Sass to compile, and do a hard reload and you'll see the the updated border and font settings shape the card component! All without having to write any custom CSS.
