Part one Customize a card with theme settings

On the opposite side of the screen you'll see a USWDS Card component. Let's customize this card with theme settings.

View available theme settings.

Visit the USWDS Card component settings table to see a list of available settings. Settings allow you to apply a global theme to USWDS components and styles without modifying CSS. Today's we're going to give our card a light golden look.

Customize border settings.

Let's start but updating the border to remove the border and give the card a rounder border radius.

We want to give our card a border radius of 2 units. To do that we'll set the value of the border radius to lg then set the value of the large border radius to 2.

description _theme/_uswds-theme.scss

@use "uswds-core" with (
  ...

  $theme-border-radius-lg: 2,
  $theme-card-border-radius: "lg",
  $theme-card-border-width: 0,

);

Customize type settings.

Next lets update the typesetting of our header to use a sans font, increase the size ("xl"), and use a tight line height (2).

description _theme/_uswds-theme.scss

@use "uswds-core" with (
  ...
  $theme-border-radius-lg: 2,
  $theme-card-border-radius: "lg",
  $theme-card-border-width: 0,
  $theme-card-header-typeset: ("sans", "xl", 2),

);

Save, wait for the Sass to compile, and reload. You'll see the the updated border and font settings shape the card component! All without having to write any custom CSS.

But maybe there are additional changes your team would like to make that are not available in theme settings. In the next section we'll take a look at adding custom styles.
