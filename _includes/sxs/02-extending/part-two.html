Part Two Customizing card using custom styles.

Using custom styles, we can customize our components beyond theme settings.

Create a custom stylesheet for our custom card component.

To help keep things organized, we'll create a custom stylesheet with styles specific to our customized card component. In later excercises, we'll use this same stylesheet to extend the card component into a new custom component. 

touch assets/css/mc-card.scss

Import the new stylesheet into USWDS custom styles.

Navigate to _theme/_uswds-theme-custom-styles.scss and import our new stylesheet into the USWDS custom styles file.

description _theme/_uswds-theme-custom-styles.scss

@use "uswds-core" as *;
  @forward "../assets/css/styles.scss";  

  @forward "../assets/css/mc-card.scss";

Once you've updated this file, you can save and close it. Since we've added our project Sass source directory to our gulpfile in the first part of the tutorial, it should already be compiling correctly!

Heads up! If you haven't integrated your project CSS and USWDS sass, return to part seven of learning how to use USWDS

Use USWDS utilities in custom styles

Now that our stylesheet is being imported, lets write some customs styles to really make this card stand out!

We'll want to use USWDS design tokens in our stylesheets using USWDS functions and mixins. To do this, lets add uswds-core to the top of our new stylesheet with the use Sass directive.

description assets/css/mc-card.scss

@use "uswds-core" as *;

Add custom class and styles

Next lets create a custom class for our personalized component and use some USWDS utilti

description _theme/_uswds-theme.scss

@use "uswds-core" with (
  ...
  $theme-card-border-color: "indigo-30",
  $theme-card-border-radius: 3,
  $theme-card-border-width: 1,
  $theme-card-font-family: "serif"
  $theme-card-header-typeset: "sans", "xl", 2.5

);

Save, wait for he Sass to compile, and do a hard reload and you'll see the the updated border and font settings shape the card component! All without having to write any custom CSS.
