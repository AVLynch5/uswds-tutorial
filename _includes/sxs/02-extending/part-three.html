Part Three Create a custom component

In this section we'll explore extending USWDS by creating a custom testimonial component not offered by the design system.

Create a the markup for a new component.

Visit _includes/sxs/02-extending/testimonial.html. This will be where we'll add the markup for our new component. Once the markup is added, it will appear under our custom card component!

Start by adding the basic skeleton of the component. Similar to card, we'll need an outer div, a body, and footer section. We'll call this component ut-testimonial.

description _includes/sxs/02-extending/testimonial.html

  <div class="ut-testimonial">
    <div class="ut-testimonial__body"></div>
    <div class="ut-testimonial__footer"></div>
  </div>

Now let's add a quote to our testimonial body.

description _includes/sxs/02-extending/testimonial.html

<div class="ut-testimonial">
  <div class="ut-testimonial__body">
    <p>
      "Wow, this component sure is great"
    </p>
  </div>
  <div class="ut-testimonial__footer"></div>
</div>

Now let's create a profile section which will feature a profile picture, name, and title.

description _includes/sxs/02-extending/testimonial.html

<div class="ut-testimonial">
  ...
  <div class="ut-testimonial__footer">
    <img src="https://doodleipsum.com/100x100/avatar-5?bg=a5a8eb&shape=circle&n=1" class="ut-testimonial__icon" alt="Profile pic">

Add stylesheet partial.

Create a new stylesheet partial for our new component. 

touch assets/css/components/_ut-testimonial.scss

Remember to import the new stylesheet into USWDS custom styles.

description _theme/_uswds-theme-custom-styles.scss

@use "uswds-core" as *;
  @forward "../assets/css/styles.scss";
  @forward "../assets/css/_ut-card.scss";

  @forward "../assets/css/_ut-testimonial.scss";

Start styling the component.

Just like using the color() function to use USWDS color tokens, we can use units() to allow native CSS properties to make use of USWDS spacing tokens. To get our base styles for our new component, add the following styles to the new stylesheet.

description assets/css/components/ut-testimonial.scss

  @use "uswds-core" as *;

  .ut-testimonial {
    background-color: color("indigo-10");
    padding: units(3);
  }

  .ut-testimonial__body p {
    text-align: center;
  }

  .ut-testimonial__footer {
    display: flex;
    flex-direction: row;
    margin-top: units(3);
  }

  .ut-testimonial img {
    @include u-circle(10);
  }

  .ut-testimonial__profile {
    margin-left: units(3)
  }

  .ut-testimonial__profile-name {
    color: color("indigo-70v");
  }

  .ut-testimonial__profile-title {
    margin-top: units(0);
  }

Can you spot all the uses of USWDS tokens?

Styling the testimonial icon.

USWDS uses variables to maintain consistent styles across different definitions. let's create a variable for our icon stroke and fill at the top of our stylesheet.

description assets/css/components/ut-testimonial.scss

@use "uswds-core" as *;


$ut-testimonial-icon-color: color("indigo-30");

.ut-testimonial__icon {
  fill: $ut-testimonial-icon-color;
  stroke: $ut-testimonial-icon-color;
}

Using USWDS utility mixins.

USWDS also offers an extensive library of utility mixins which you can use to simplify style setting. Let's use some of the typeset mixins to customize our font weight and size. It's standard convention to include your mixins at the top of each style definition.

description assets/css/components/ut-testimonial.scss

.ut-testimonial__body p {

  @include u-font-size("heading", "xl");
  font-weight: font-weight("bold");
  text-align: center;
}

...

.ut-testimonial__profile-title {
  margin-top: units(0);

  font-size: font-size("body", 5)
}

Using USWDS theme tokens with mixins.

Finally, let's use a combination of utility mixins and uswds theme settings to display how you can add uniformity across your project! In this example, we'll add the border settings from step one to help our new component sit nicely with the original card.

description assets/css/components/ut-testimonial.scss

.ut-testimonial {
  @include u-margin-x(1);
  @include u-border($theme-card-border-width);
  @include u-radius($theme-card-border-radius);
  background-color: color("indigo-10");
  padding: units(3);
}

And voila! You've used theme settings, custom styles, USWDS tokens, and utility mixins to theme, customize, and extend USWDS into creating our very own component! Great work!
